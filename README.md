# Project Title

Dimension Extractor

## Project Explanation
This project is designed to extract dimensions from various types of files. The main purpose is to automate the process of extracting dimensions, which can be tedious and error-prone if done manually. 

## Setup Instructions
1. **Clone the repository**: 
   ```bash
   git clone https://github.com/Prashanthdesai2603/dimension_extractor.git
   cd dimension_extractor
   ```
2. **Install dependencies**: 
   You need to have Python 3.x installed. Then, run:
   ```bash
   pip install -r requirements.txt
   ```

## How to Run the Project
To run the project, use the following command:
```bash
python main.py
```
Make sure to replace `main.py` with the name of your main script if it's different.
